အရေးအကြီးဆုံးအချက်: 
header() သုံးပြီးတာနဲ့ အောက်မှာ exit; သို့မဟုတ် die(); ကို အမြဲတမ်း လိုက်ရေးရပါမယ်။ 
ဘာလို့လဲဆိုတော့ Browser ကို နောက် page ကိုသွားဖို့ ညွှန်ကြားလိုက်ပေမယ့်၊ 
Server ပေါ်က PHP ကတော့ အောက်က ကျန်နေတဲ့ code တွေကို ဆက် run နေတတ်လို့ပါ။

=================

ဘာတွေ ရှောင်ရမလဲ (What to skip/avoid)
→ header() မရေးခင် ဘာ HTML element, space, သို့မဟုတ် echo မှ မရေးရပါဘူး။ 
ရေးမိရင် PHP က "Headers already sent by..." ဆိုတဲ့ နာမည်ကြီး Error ကို ပြပါလိမ့်မယ်။
